# LetWin

从微软服务器获取最新的原版 Windows 系统镜像

## 技术栈

- **React 18** - 前端框架
- **TypeScript** - 类型安全
- **Vite** - 构建工具
- **Tailwind CSS v4** - 样式框架
- **Coss UI** - 组件库（基于 Base UI）
- **Lucide Icons** - 图标库
- **React Router** - 路由管理

## 功能特性

- 查看最新的 Windows 11 和 Windows 10 系统镜像
- 支持按系统版本、语言、版本类型筛选
- 显示镜像详细信息（大小、SHA1、架构等）
- 直接从微软服务器下载
- 复制下载直链

## 开发

安装依赖：

```bash
pnpm install
```

启动开发服务器：

```bash
pnpm dev
```

构建生产版本：

```bash
pnpm build
```

预览生产构建：

```bash
pnpm preview
```

本项目使用 LetWin API 获取 Windows 镜像信息。
